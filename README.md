# MarcaPonto

Sistema django para registro de jornada de trabalho.

clonar projeto

cd MarcaPonto/

*ambiente linux:* 
source venv/bin/active

*ambiente windows:* 
python3 -m venv venv
venv/Scripts/active

pip install django

*para criar usuario para logar e gerenciar*
python manage.py createsuperuser

*rodar aplicação*
python manage.py runserver

Acesso: http://127.0.0.1:8000/
